
                                            ΣΧΕΔΙΑΣΜΟΣ/ΣΚΕΠΤΙΚΟ ΣΥΝΘΕΣΗΣ ΣΤΟ SC


Η κάθε ηχητική σύνθεση,θα γίνει στο Super Collider.
Σε επίπεδο σχεδιασμού, υπάρχουν 3 βασικά αλληλεξαρτώμενα πράγματα:

- Πηγές 
- Παράγωγα
- Ιεράρχηση(χρονική)

 Οταν και αν αυτά επιτευχθούν, τότε υπάρχει μία Σύνθεση.


                                                ΠΗΓΕΣ

  Οι πηγές αφορούν κάθε είδους δομή/πρότυπο, από την οποία μπορούν να προκύψουν διάφορα ηχητικά αποτελέσματα
ανάλογα με την επεξεργασία των παραμέτρων της.
 Οι παράμετροί των πηγων ελέγχουν ένα η περισσότερα από διάφορα βασικά χαρακτηριστικά όπως:

- Ύψος του ήχου(frequency),ενταση(amplitude),διάρκεια(duration),φάση(phase),μετατόπιση αρχικού σήματος(dcBias)

...αλλά και άλλα, πιο εξειδικευμένα.

 Στο SC, οι πηγές αντιστοιχούν σε δομές που επιτρέπουν την προσεγμένη οργάνωση παραμέτρων,όπως τα synthDefs, 
ή σε πιο απλές δομές(Functions).



                                                ΠΑΡΑΓΩΓΑ 

  Τα παράγωγα αντιστοιχούν σε τμήματα ήχων, που προκύπτουν από τις πηγές. Είναι οι ηχητικές δοκιμές των πηγών που φτιάχτηκαν.

  Aν οι πηγές είναι απλές functions, τότε τα παράγωγα είναι μεταβλητες που εκτελούν την μέθοδο play( πχ y = ~f.play)
  Αν οι πηγές είναι synthDefs, τα παράγωγα είναι synths,διάφορα μοτίβα(patterns), ή ακόμα και ομάδες από Synths(Groups).
  Η ποικιλία τους εξαρτάται από το είδος των πηγών, και το πόση τροποποίηση των αρχικών παραμέτρων αυτές επιτρέπουν.
  Γίνονται δοκιμές, μέχρι να προκύψουν επιθυμητά αποτελέσματα.



                                                ΙΕΡΑΡΧΗΣΗ


   Αφού έχουν φτιαχτεί διάφορες πηγές ήχου(SynthDefs), και μέσω δοκιμών προκύπτουν ενδιαφέροντα παράγωγα(Synths/Patterns/Groups),
   τότε μένει η σύνθεση, το πως αυτά θα δημιουργήσουν ένα μουσικό/ατμοσφαιρικό σύνολο, ή ένα ηχητικό περιβάλλον.
   Δομές όπως oι Routines, επιτρέπουν την κατάταξη και χρονική ιεράρχηση των μεμονωμενων τμηματων ήχου.


   Οπότε, εν τέλει η Συνθεση, στο τεχνικό επίπεδο του SC, θα μοιάζει κάπως έτσι:


  _______________________________________________________________________________________________________________--

                                                  
                                ΔΕΙΓΜΑ-ΣΚΕΛΕΤΟΣ ΤΕΛΙΚΗΣ ΜΟΡΦΗΣ ΣΥΝΘΕΣΗΣ


(
SynthDef.new({ \source1,
   ....
}).add;
)     


(
SynthDef.new({ \source2,
   ....
}).add;
)     


(
SynthDef.new({ \source3,
   ....
}).add;
)     

------------------------------------------------

(
~synth1 =(\source1, []);
)  

(
~synth2 =(\source2, []);
)   
  

(
~pattern1 = Pbind({
       \instrument,  \source1,
});
)    

(
~pattern2 = Pbind({
       \instrument,  \source2,
});
)                              


-----------------------------------------------------------------------------
(
  ~sequence = Routine.new({
                  ~synth1.play;
                  value.yield;
                  ~pattern1.play;
                  value.yield;
                  ~synth2.play;
                  value.yield;
                  ~pattern2.play;
                  value.yield;
               
                    ...etc..
});
)


   
~sequence.play;
